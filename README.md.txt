# 📊 Sales Data Analysis

Analyze a dataset of 500 sales transactions including product categories, regions, and total revenue.

## 📁 Project Structure
- `data/`: Raw sales dataset
- `notebooks/`: Jupyter notebook for analysis
- `images/`: Graphs and charts
- `README.md`: Project overview

## 📈 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Key Insights
- Top performing product categories
- Regional sales breakdown
- Monthly sales trends
